Troubleshooting-log-21.md

Issue: User unable to access SharePoint site.
Impact: Medium – project documents blocked.

Diagnosis
403 Forbidden
User not included in SharePoint permissions group
Other users OK

Actions
Added user to SharePoint "Members" group.
Synced permissions in Microsoft Admin Centre.
Tested access → success.

Resolution
User accessed site instantly.

Root Cause
Missing permission assignment.
